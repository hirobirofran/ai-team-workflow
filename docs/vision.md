# ビジョン：AI チームで開発を加速する

## 解決したい問題

- AI が長い作業で混乱・暴走する（コンテキスト爆発）
- エラーが出ると荒れ狂い、修正指示で更に混乱する
- 要件の把握が苦手、推測で断定して突き進む
- セッションをまたぐと文脈が失われる
- 単一 AI では得意不得意をカバーできない

## 実現したいこと

複数の AI を**人間のチームのように役割分担**させ、それぞれの得意分野を活かしながら
コンテキスト爆発・暴走・ケアレスミスを減らした開発フローを確立する。

```text
アイデア → リサーチ → 要件整理 → 設計 → 実装 → レビュー → 改善
  人間      NotebookLM   Gemini    Gemini  Claude   Gemini    人間+AI
                         (PM役)   (設計役)  Code    (確認役)
```

## 適用範囲

- **開発用途**：Terraform/Terragrunt、バックエンド開発、インフラ構築など
- **個人用途**：Garmin データ分析、タスク管理、メール自動化など
- **汎用**：どのリポジトリ・どの AI ツールでも使い回せるテンプレート集として

## 成功の定義

- 1 日中 AI と格闘して消耗することがなくなる
- AI が迷子になったとき、すぐ軌道修正できる
- Claude Code が使えないときも他の AI でしのげる
- テンプレートとナレッジが蓄積され、使うたびに改善される
