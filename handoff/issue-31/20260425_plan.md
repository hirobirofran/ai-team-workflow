## サブイシュー Body / initial_prompt の方針

各サブ Issue の body は CLAUDE.md「整理ハードル放棄パターン」（コンパクト本文 + 根拠ファイル参照）に従う。長く書かない。initial_prompt は B1/B2 stub 構造を踏襲（forcing function 待ちでなく即着手前提）:

- 起動コマンド（コピペ可）
- 起動条件（即着手 OK、依存タスクの完了確認）
- 必読ファイル（CLAUDE.md / Issue 本文 / 該当根拠ファイル）
- タスクの性質（PR 必須 or main 直 push 可）
- 重要な制約（コミット粒度・分割推奨等）
- 終了条件（PR マージ or commit、Issue close、AAR 保存）
- 開始手順
